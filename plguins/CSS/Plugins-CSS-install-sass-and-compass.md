# 安裝 sass 及 compass

## 使用 homebrew 安裝 ruby

```sh
brew install ruby
```

安裝完成後，重新開啟新的 terminal 視窗，以便讀取到新的 ruby 路徑

## 安裝 sass

```sh
gem install sass
```

## 安裝 compass

### 更新gem系統

```sh
gem update --system
```

### 安裝

```sh
gem install compass
```
